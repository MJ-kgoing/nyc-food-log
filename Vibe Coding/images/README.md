Put your venue photos in this folder.

Example: save a file as `lartusi.jpg`, then in `script.js` set:

`image: "./images/lartusi.jpg"`

Supported formats: JPG, PNG, WebP, GIF (whatever your browser can display).

## Site background image

To use a background image for the whole site, add a file named:

- `background.jpg`

into this `images/` folder (or change `--bg-image` in `styles.css` to match your filename).
